## Browser-based PDF editor

After searching the web, I could not find a free-to-modify, open-source, browser-based PDF editor. So I built one! It is very buggy, so feel free to report any bugs that may come up.

# Usage

It is a React app, so you can simply use it as is.

# Modifications

Core functionality is in pdf.js, shapes.js, and pdf.css. Adding images does not work without a functional backend (example provided in the backend folder).
